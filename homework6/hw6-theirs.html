Problem 5: Creating a dictionary

 

To create an empty dictionary, you can declare it like so:

 

"dictionary = {}"

 

A dictionary contains keys and values that make a "key-value" pair. 

The key is the string used for indexing and the value is what the key refers to.

 

Here is an example of how to create a dictionary with key-value pairs already populated with elements:

 

"dictionary = {"Alice": 2341, "Beth": 9102, "Cecil": 3250}"

Problem 6:  Adding key-value pairs to a dictionary

 

To add key-value pairs to a dictionary, you need to first write the dictionary name and then the key name inside 

brackets next to the dictionary name, then have it equal to the value of that key, like so:

 

#first name and declare the dictionary

dictionary = {}

 

#Call the dictionary with a key name, then have it equal to a value.

dictionary["a"] = 2

 

#This will create a dictionary that looks like:

dictionary = {"a": 2}
